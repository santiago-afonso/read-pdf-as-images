Minimal usage prompt for read-pdf-as-images

- Task: visually inspect specific PDF pages (no OCR/text extraction).
- Command to run:
  scripts/read-pdf-as-images "<pdf_path>" --pages "<comma/ranges>"
- Defaults: PNG, 220 DPI. Output files under tmp/pdf_renders/<pdf-name>/page-###.png
- The command writes JSONL to stderr with {"page", "path", ...}; attach each image by the emitted path.

Example
  scripts/read-pdf-as-images "Input/Classifiers/FY25 Theme Taxonomy. Mar 2025.pdf" --pages "1-3,6-8"
